# 🎭 Moodify – Feel Better Instantly

##  Description
Moodify is a web application that provides personalized recommendations based on the user’s mood. Users can select how they are feeling (happy, sad, stressed, etc.), and the app will suggest movies, songs, and motivational quotes to improve their mood.

Additionally, users can:
- Save their favorite content
- Write personal notes
- Interact with a simple chatbot interface

The goal of the application is to create a positive and engaging experience that helps users feel better through content recommendations.

---

## 🔗 APIs Used
- OMDB API → Movie suggestions  
- ZenQuotes API → Motivational quotes  
- iTunes API → Song recommendations with preview  

---

##  Planned Features
- Mood-based recommendation system  
- Search functionality for movies and songs  
- Filter results based on mood/type  
- Sort results (alphabetical / year)  
- Add/remove favorites (using localStorage)  
- Song preview player  
- Motivational quotes display  
- Notes section for user thoughts  
- Chatbot interface (basic interaction)  
- Dark/Light mode toggle  
- Fully responsive design
